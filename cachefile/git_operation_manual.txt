测试SSH连接：
ssh -T git@github.com
初始化仓库：
git init
clone远程至本地：
git clone git@github.com:one1t/mygit.git
添加文件：
git add cachefile/
提交文件：
git commit -m "注释"
推送文件：
git push -u origin master
拉取文件：
git pull origin master
查看远程仓库详细信息:
git remote -v
删除orign仓库（比如名称错误）:
git remote remove orign
仓库地址：重新添加远程仓库地址:
git remote add origin 
 提交到远程仓库的master主干:
gti push -u origin master
